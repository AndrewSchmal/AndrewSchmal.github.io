Client Project: Starter Guide for ZZZ Game

1. Project Overview

  • Purpose: My client project is going to be a beginner's guide website to the gacha game Zenless Zone Zero. The client will be my friend Andrew Meadows. When I asked him if he had something he wanted me to make, he told me he would like something like a "starter's guide" for ZZZ so he could catch up. Essentially, basic info like how to do dailies, teambuilding tips, and something related to resource farming (which many gacha have) so he knows what materials he should prioritize farming for. The guide will provide structured, easy-to-navigate information and interactive tools to help players optimize their gameplay and track progress. The purpose extends beyond the client, as the website will be useful for any player looking to learn more about ZZZ.
  • Intended Users: Any new or casual ZZZ players seeking beginner guides, character optimization tips, or farming strategies. My client Andrew fits under this description, but others looking for ZZZ resources will find it useful as well.
  • Content: The website will feature:
    • Dynamic Tools:
      • A material calculator for upgrading characters and skills.
        • Includes sliders to adjust the starting level and see required resources.
        • Breaks down resource costs by levels or milestones (e.g., max skill level).
      • A farming planner to help players optimize daily or weekly in-game tasks.
        • Includes priority recommendations based on resource scarcity and character needs.
        • Tracks completion status for tasks like dailies, expert challenges, and notorious hunts.
    • Starter Guides: Beginner-friendly explanations of core mechanics.
      • How to efficiently complete daily tasks.
      • Understanding the role of team synergy in battles.
      • Overview of game modes and how to approach them effectively.
      • Introduction to in-game currencies and how to use them wisely.
    • Tips and Strategies for Leveling Up Faster:
      • Optimal resource allocation for new players.
      • What to do for your "dailies".
      • Recommended farming routes and routines.
    • Character-Specific Optimization Guides:
      • Examples like "Mika's Ultimate Skill Upgrade Path."
      • Resource farming planner for specific characters.
      • Role-based team composition recommendations for each character.
    • Interactive and Responsive Elements:
      • Clean and user-friendly UI, especially for dynamic tools like calculators and planners.
